# Treatment - בוט תזכורת תרופות

**ריפו:** [amirbiron/Treatment](https://github.com/amirbiron/Treatment)
**סטטוס:** פעיל (בייצור)

בוט טלגרם חכם לתזכורת תרופות עם מעקב מלאי, יומן תופעות לוואי, מצב מטפל, ודוחות — מותאם לזקנים ומטפלים.

## פיצ'רים מרכזיים
- **תזכורות חכמות** — תזכורות אוטומטיות עם snooze ותזכורות חוזרות
- **מעקב מלאי** — ספירת כדורים עם התראות מלאי נמוך
- **יומן תופעות לוואי** — רישום יומי עם מעקב מצב רוח
- **מצב מטפל** — דיווחים אוטומטיים למטפלים ובני משפחה
- **דוחות תקופתיים** — דוחות שבועיים ויומיים
- **ממשק מותאם לזקנים** — כפתורים גדולים, עברית מלאה

## Tech Stack
Python, python-telegram-bot, SQLAlchemy, aiosqlite, APScheduler, tornado, GitHub Actions CI/CD

## מה עשיתי אני
פיתוח מלא מאפס — ארכיטקטורה מודולרית, מודלי DB, handlers מופרדים, מנוע תזמון, בדיקות, ו-CI/CD.
