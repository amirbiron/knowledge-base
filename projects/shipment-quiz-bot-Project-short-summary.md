# Shipment Quiz Bot - בוט חידון לטלגרם

**ריפו:** [amirbiron/shipment-quiz-bot](https://github.com/amirbiron/shipment-quiz-bot)
**סטטוס:** פעיל

בוט טלגרם לתרגול ולמידה של פרויקט Shipment-bot באמצעות שאלות רב-ברירה ונכון/לא נכון.

## פיצ'רים מרכזיים
- **6 קטגוריות** — Deployment, Architecture, Database, State Machine, API, Testing
- **מצבי משחק** — בחירת קטגוריה ספציפית או שאלה רנדומלית
- **הסברים מפורטים** — הסבר לכל תשובה נכונה ושגויה
- **טעינה דינמית** — שאלות נטענות מ-GitHub Gist בלי deploy חדש
- **Gemini AI** — אינטגרציה עם Google Gemini להעשרת תוכן

## Tech Stack
Node.js, Telegraf, Google Generative AI (Gemini), ES Modules, dotenv

## מה עשיתי אני
פיתוח מלא מאפס — עיצוב מבנה החידון, כתיבת השאלות, אינטגרציה עם Telegram ו-Gemini AI, ומנגנון טעינה דינמית מ-Gist.
