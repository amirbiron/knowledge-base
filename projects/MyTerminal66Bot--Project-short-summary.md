# MyTerminal66Bot

**ריפו:** [amirbiron/MyTerminal66Bot-](https://github.com/amirbiron/MyTerminal66Bot-)
**סטטוס:** פעיל

בוט טלגרם להרצת קוד ופקודות Shell מרחוק – תומך Python, JavaScript, Java ו-Bash עם ממשק Web App אינטראקטיבי.

## פיצ'רים מרכזיים
- הרצת קוד Python, JavaScript, Java ו-Shell דרך טלגרם
- Web App טלגרם עם טרמינל PTY אינטראקטיבי (WebSocket)
- ניהול הרשאות מבוסס OWNER_ID ורשימת פקודות מותרות
- ספריות מדעיות מותקנות מראש (numpy, matplotlib, pygame)
- Task Manager מובנה ו-Activity Reporter ל-MongoDB
- Docker image מוכן עם כל התלויות

## Tech Stack
Python 3.11, python-telegram-bot, Flask, flask-sock (WebSocket), Gunicorn, Docker, MongoDB
