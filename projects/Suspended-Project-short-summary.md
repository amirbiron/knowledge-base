# Suspended - בוט ניטור והשעיה חכמה ל-Render

**ריפו:** [amirbiron/Suspended](https://github.com/amirbiron/Suspended)
**סטטוס:** פעיל (בייצור)

בוט טלגרם שמנטר שירותי Render, משעה אוטומטית שירותים לא פעילים לחיסכון בעלויות, ומספק ניטור לוגים בזמן אמת.

## פיצ'רים מרכזיים
- **ניטור אוטומטי** — בדיקה יומית של פעילות בוטים על Render
- **השעיה חכמה** — השעיית שירותים שלא היו בשימוש (מספר ימים מוגדר)
- **התראות דיפלוי** — זיהוי מעברי סטטוס בזמן אמת
- **ניטור לוגים** — זיהוי אוטומטי של שגיאות רגילות וקריטיות
- **צפייה בלוגים** — הצגת לוגים ישירות מהבוט עם סינון
- **מעקב פעילות** — היסטוריה מלאה ב-MongoDB

## Tech Stack
Python, python-telegram-bot, MongoDB, Render API, schedule, python-dotenv

## מה עשיתי אני
פיתוח מלא מאפס — ארכיטקטורה, Backend, אינטגרציה עם Render API ו-Telegram Bot API, מערכת ניטור לוגים, ומנגנון השעיה/החזרה אוטומטי.
