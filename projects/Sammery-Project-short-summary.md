# Sammery - בוט סיכומים שבועיים לטלגרם

**ריפו:** [amirbiron/Sammery](https://github.com/amirbiron/Sammery)
**סטטוס:** פעיל (בייצור)

בוט טלגרם שמאזין לפוסטים בערוץ, שומר אותם ב-MongoDB, ויוצר סיכומים שבועיים אוטומטיים באמצעות GPT-4 Turbo.

## פיצ'רים מרכזיים
- **סיכומים חכמים** — GPT-4 Turbo יוצר סיכומים קוהרנטיים ומעניינים
- **תזמון גמיש** — בחירת שעת פרסום ביום שישי עם ממשק כפתורים
- **תצוגה מקדימה** — צפייה בסיכום לפני פרסום עם אפשרות ליצור מחדש
- **אכלוס היסטוריה** — Forward פוסטים ישנים לבוט לשמירה
- **פרסום אוטומטי** — תמיכה ב-AUTO_PUBLISH_ON_START

## Tech Stack
Python, python-telegram-bot, OpenAI GPT-4, MongoDB, Flask, Gunicorn, Docker

## מה עשיתי אני
פיתוח מלא מאפס — ארכיטקטורה, אינטגרציה עם GPT-4 ו-Telegram, מנגנון תזמון, פריסה על Render עם Docker.
