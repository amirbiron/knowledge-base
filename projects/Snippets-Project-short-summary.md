# Snippets - אוסף קטעי קוד Python

**ריפו:** [amirbiron/Snippets](https://github.com/amirbiron/Snippets)
**סטטוס:** פעיל

אתר סטטי המציג אוסף מקיף של קטעי קוד Python מאורגנים בקטגוריות, עם כלי המרה אוטומטי מ-Markdown ל-HTML.

## פיצ'רים מרכזיים
- **עשרות סניפטים** — קטעי קוד Python מאורגנים בקטגוריות נושאיות
- **העתקה בלחיצה** — Copy to Clipboard לכל קטע קוד
- **כלי המרה** — סקריפט Python להמרת Markdown ל-HTML עם offset אוטומטי
- **ממשק מודרני** — עיצוב רספונסיבי עם אייקונים (Font Awesome)

## Tech Stack
HTML5, CSS3, JavaScript (Vanilla), Python (כלי המרה)
